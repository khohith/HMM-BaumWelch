# Hidden Markov Model using Baum-Welch Algorithm

Name: KARRI KHOHITH


Register Number: TCR24CS041


Course: PATTERN RECOGNITION

## Description
This project implements a Hidden Markov Model (HMM) using the Baum-Welch Algorithm.
A visual interface is created using Streamlit to display the trained transition and emission matrices.

## How to Run
1. Install dependencies:
   pip install numpy streamlit matplotlib

2. Run:
   streamlit run app.py
